PROJECT TITLE: Object Finder 

PURPOSE OF PROJECT: To program a robot that finds an object within a predefined area and push that object.
VERSION or DATE: 24/03/22
AUTHORS: Sicheng xie.
USER INSTRUCTIONS: 
Define an rectangular area using black tape.
Place the robot in one corner with the colour sensor over the black tape, and with the wheels paralell to the tape.
Press Enter when prompted to begin the program. 


Driver.Java: Main method, contains inittialisation of all variables, sensors, pilot, behaviours and arbitrator.

Find: Behaviour for finding an object in range.

Push: Behaviour for pushing when the robot finds an object.

Mapping: This behaviour maps the area and provides details for to be able to stay on the map.

Clapping: The behaviour for starting and continuing the robot.
